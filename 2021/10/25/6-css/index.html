2.2层次选择器

1.后代选择器：在某个元素的后面

/*  后代选择器 */
body p{
    background: aqua;
}

2.子选择器:,一代 

/*子选择器*/
body>p{
    background: aqua;
}

3.相邻选择器 同辈(对下不对上 )

/*相邻选择器 只有一个，相邻（向下）*/
.active + p{
    background: aqua;
}

4.通用选择器

/*通用兄弟选择器，当前选中元素的向下的所有兄弟元素*/
.active~p{
    background: #20d071;
}

2.2结构伪类选择器

伪类：条件 

/*ul的第一个子元素*/
ul li:first-child{
    background: #20d071;
}
/*ul的最后一个元素*/
ul li:last-child{
    background: aqua;
}
/*  选中p1:定位到父元素，选择当前的第一个元素
选择当前p元素的父级元素，选中父级元素的第一个，并且是当前元素才生效！
*/
p:nth-child(2){
    background: #20d071;
}
/*选中父元素，下的p元素的第二个类型*/
p:nth-of-type(2){
    background: aqua;
}

5.3父级边框塌陷的问题

clear:left; 左侧不允许有浮动元素
clear:right;右侧侧不允许有浮动元素
clear:both; 两侧侧不允许有浮动元素
clear: none;

解决方案：

1.增加父级元素的高度

简单，元素假设有了固定的高度，就会被限制

2.增加一个空的div标签，清除浮动

简单，代码中尽量避免空div

<div class = "clear"></div>

.clear{
    clear:both;
    margin:0;
    padding:0;
}

3.overflow

简单，下拉的一些场景避免使用

在父级元素中增加overflow:hidden;

4.父类添加一个伪类：after

推荐，没有副作用

#father:after{
    content '';
    display:block;
    clear:both;
}
